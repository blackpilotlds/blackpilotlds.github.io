Using CSS Keyframe Animations

by Sergei Aleinikov Spring 2020 WDD 230-10 Web Frontend Development I

JavaScript or CSS

Usually animation is the part of JavaScript BUT
with modern browsers we can animate in CSS[1]

[1]“CSS - Keyframe Animations.” CSS - Keyframe Animations, Badell Maman, 19 Nov. 2016, youtu.be/AGFXMU8rink.

@keyframes

"The @keyframes CSS at-rule, controls the intermediate steps in a CSS animation sequence by defining styles for keyframes (or waypoints) along the animation sequence. This gives more control over the intermediate steps of the animation sequence than transitions."[2]

[2]“@Keyframes.” MDN Web Docs, developer.mozilla.org/en-US/docs/Web/CSS/@keyframes.
